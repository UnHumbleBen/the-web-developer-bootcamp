# Databases

## Intro to Databases
* What is it?
  * collection of information/data
  * has an interface
    * MongoDB: `db.dogs.find()`
* SQL(relational) vs. NoSQL(non-relational)
  * Relational:
    * Tabular
    * Not flexible (requires exact patterns for each entry)
  * Non-relational:
    * Nested
    * Flexible (but not necessarly better than relational)
